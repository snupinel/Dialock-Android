# DailySummary (일일 요약 앱)

**DailySummary**는 하루를 기록하고 요약할 수 있는 Android 앱입니다.  
달력 기반의 UI를 통해 날짜별로 일기를 작성하고, 한눈에 요약을 확인할 수 있습니다.
